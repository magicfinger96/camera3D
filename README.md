# Caméra 3D
![alt text](https://github.com/magicfinger96/camera3D/blob/master/img/screen.PNG)
Pour compiler: make
Pour executer: ./exec

Les commandes:
pour déplacer la caméra: q z d s
pour zoom in/out: x w
pour pivoter: les fleches du clavier
